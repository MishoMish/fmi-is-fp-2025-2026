## Задача 1: Ехо програма

Напишете програма, която чете редове от конзолата и ги печата обратно (обърнати), докато потребителят не напише `"quit"`.

```haskell
main :: IO ()
```

```
> hello
olleh
> world
dlrow
> quit
Довиждане!
```

Напишете интерактивен калкулатор, който чете операция (`+`, `-`, `*`, `/`) и две числа:

```

Второ число: 4
```

## Задача 4: CSV четене

Напишете функция, която чете CSV файл и го отпечатва форматирано като таблица:

```haskell
printCSV :: FilePath -> IO ()
```

Входен файл `data.csv`:
```
Име,Възраст,Град
Иван,25,София
Мария,22,Пловдив
```

Изход:
```
| Име   | Възраст | Град    |
|-------|---------|---------|
| Иван  | 25      | София   |
| Мария | 22      | Пловдив |
```

## Задача 5: TODO списък

Напишете програма за управление на TODO списък:
- `add <задача>` — добавя задача
- `done <номер>` — маркира като завършена
- `list` — показва всички задачи
- `save` — записва във файл
- `load` — зарежда от файл
- `quit` — изход

```haskell
main :: IO ()
```
