## –ó–∞–¥–∞—á–∞ 2: `encode` / `decode` (Run-Length Encoding)

```haskell
encode :: Eq a => [a] -> [(Int, a)]
decode :: [(Int, a)] -> [a]
```

```haskell
>>> encode "aaabbcccc"
[(3,'a'),(2,'b'),(4,'c')]
>>> decode [(3,'a'),(2,'b'),(4,'c')]
"aaabbcccc"
```
