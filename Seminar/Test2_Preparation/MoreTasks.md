## Задача 1 — Основи и функции от по-висок ред

**а)** Имплементирайте функция `myFoldr :: (a -> b -> b) -> b -> [a] -> b`, без да използвате `foldr` или `foldl`.

---

**б)** Имплементирайте функция `myZipWith :: (a -> b -> c) -> [a] -> [b] -> [c]`, без да използвате вградени функции освен pattern matching и рекурсия.

---

**в)** Напишете функция `applyN :: Int -> (a -> a) -> a -> a`, която прилага дадена функция `n` пъти върху начална стойност.

Пример: `applyN 3 (+10) 0 == 30`

---

**г)** Напишете функция `myIterate :: (a -> a) -> a -> [a]`, която генерира безкраен списък, като прилага функцията последователно.

Пример: `take 5 (myIterate (*2) 1) == [1,2,4,8,16]`

---

**д)** Напишете функция `composeAll :: [a -> a] -> a -> a`, която прилага всички функции от списъка последователно **отдясно наляво** (като `(.)`).

Пример: `composeAll [(+1), (*2), (+3)] 4 == 15` _(т.е. ((4+3)\*2)+1)_

---

**е)** Напишете функция `myUnfoldr :: (b -> Maybe (a, b)) -> b -> [a]`, която генерира списък от начална seed стойност, докато функцията върне `Nothing`.

Пример: `myUnfoldr (\n -> if n == 0 then Nothing else Just (n, n-1)) 5 == [5,4,3,2,1]`

## Задача 3 — Списъци и операции

## Задача 5 — Работни задачи и тикет номера

- **Заглавие** (`String`)

- **Краен срок** (`Deadline`)
- **Подзадачи** — списък от работни задачи от тип `WorkItem` (може да е празен)

> **Формат на тикет номер:** `(2 главни латински букви = код на проект)(4 цифри = година)(4 цифри = пореден номер)` — общо **10 символа**.  
> Пример: `"AB20260001"` — проект AB, 2026, задача 0001.  
> За **критични** задачи поредният номер е между `5000` и `9999`.  
> За **обикновени** задачи поредният номер е между `0001` и `4999`.

---

> **Упътване:** Трябва да направите `WorkItem` инстанция на `Eq`. Помислете кое поле е подходящ уникален ключ.

---

**е)** Напишете функция `grandparentTask :: Eq WorkItem => WorkItem -> WorkItem -> Maybe WorkItem`, която използва резултата от предходната подточка.

---

**ж)** Напишете функция `greatGrandSubtasks :: WorkItem -> [WorkItem]`, която връща всички подзадачи на дълбочина **точно 3** от дадения елемент.

**з)** Напишете функция, която приема списък от `WorkItem` и **присвоява валиден уникален тикет номер** на всяка задача и на всичките й подзадачи. Поредният номер в края започва от `0001` и нараства с всяко ново присвояване. Приемете, че всеки тикет номер, който вече има стойност, е валиден.
